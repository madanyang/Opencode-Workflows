# Notion MCP

Official Notion server for workspace access.

## Installation

```jsonc
{
  "mcp": {
    "notion": {
      "type": "remote",
      "url": "https://notion-mcp.notion.so",
      "oauth": {}
    }
  }
}
```

## Setup

```bash
opencode mcp auth notion
```

Opens Notion OAuth authorization flow.
