# Opencode Workflows

A collection of Opencode-based command templates and workflow patterns for building sophisticated command-based projects.

## Currently Included

### Commands2Skills Template

A universal template for creating Opencode-based projects with:

- **Command Integration Pattern**: 4-part architecture for command-to-tool interfaces
- **Template Structure**: Organized framework for command-based projects
- **Documentation System**: Automated maintenance via specialized agents
- **Model Compatibility**: Tested with GPT-5.1 series models
- **Example Commands**: Repository exploration and mathematical calculation tools

### Getting Started

1. Clone this repository locally to your machine
2. Review the Commands2Skills template structure and components
3. Select and copy the elements you want to include in your project
4. Create your own `opencode.json` based on the `example-opencode.json` configuration

See `Commands2Skills/README.md` for comprehensive documentation on:

## About

This repository provides tested Opencode command patterns and workflow templates that demonstrate best practices for creating maintainable, scalable command-based projects with external tool integration.
